# Linux Production Server Project

## Project Overview
This project demonstrates a real-world Linux Administration setup using Kali Linux. It includes Apache web server setup, user and group management, firewall configuration, backup automation, cron jobs, and system monitoring.

## Features
- Apache Web Server Setup
- User and Group Management
- File Permission Management
- UFW Firewall Configuration
- Automated Backup Script
- Cron Job Automation
- System Monitoring Commands
- Apache Log Monitoring

## Technologies Used
- Kali Linux
- Apache2
- Bash Scripting
- UFW Firewall
- Cron Jobs
- Linux Commands

## Project Files
- backup.sh - Automated website backup script
- monitoring-commands.txt - Linux monitoring commands
- firewall-rules.txt - Firewall configuration rules
- users.txt - User and group management commands

## Backup Script
The backup script creates a compressed backup of the website directory and removes backups older than 7 days.
